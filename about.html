자유롭고 열린
제주어사전

www.jejuopen.kro.kr
💡 옛한글(아래아) 쉽게 입력하기
제주어를 검색할 때 필수적인 옛한글을 키보드로 아주 쉽게 입력할 수 있습니다.

• ㅏㅏ (모음 'ㅏ'를 두 번) 누르면 아래아(ㆍ)가 입력됩니다.
• ㅑㅑ (모음 'ㅑ'를 두 번) 누르면 쌍아래아(ᆢ)가 입력됩니다.

시작의 이유

오래전부터 제주어에 푹 빠져 있었고, 이 아름다운 언어를 담아낼 사전 구축을 여러 번 시도해 왔습니다.

최근 공식 제주어 사전인 '제주어왓'이 공개되었지만, 사용하기에 다소 불편함이 있었고 무엇보다 귀중한 데이터들이 대중에게 열려있지 않았습니다. 이에 누구나 쉽게 접근하고 데이터를 활용할 수 있는 진정한 의미의 '자유롭고 열린' 제주어 생태계를 직접 만들고자 이 프로젝트를 시작하게 되었습니다.

원본 데이터 개방 & API

사전의 이름처럼, 이 프로젝트가 가진 모든 데이터와 기능은 자유롭게 열려 있습니다. 제가 직접 수집하고 크롤링하여 정제한 전체 원본 데이터(JSON)는 GitHub 레포지토리(JejuOpen)에서 누구나 제한 없이 확인하고 다운로드하실 수 있습니다.

또한 개발자나 연구자가 이 사전을 활용해 새로운 앱이나 서비스를 만들 수 있도록 API 역시 개방해 두었습니다.

API 사용 방법 및 파라미터 확인하기
제주어 사전 API는 누구나 쉽게 호출하여 사용할 수 있는 JSON 포맷으로 결과를 반환합니다. CORS 제한이 없어 프론트엔드 프로젝트에서도 직접 호출이 가능합니다.

기본 요청 URL (GET) 
https://www.jejuopen.kro.kr/api.html
파라미터 (Parameters)
  • q : 검색어 (생략 시 사전 전체 데이터를 페이징하여 반환합니다)
  • exact : true 입력 시 뜻풀이를 제외하고 표제어/표준어가 정확히 일치하는 결과만 반환합니다.
  • page : 요청할 페이지 번호 (기본값: 1)
  • limit : 한 페이지당 불러올 데이터 개수 (기본값: 20, 최대 100)
호출 예시 
// 1. '감자'를 포함하는 단어 검색 (한 페이지에 10개씩)
https://www.jejuopen.kro.kr/api.html?q=감자&limit=10

// 2. 표준어가 '고구마'와 '정확히 일치'하는 제주어 찾기
https://www.jejuopen.kro.kr/api.html?q=고구마&exact=true

데이터의 가치

언어는 단순한 의사소통 도구를 넘어, 그 지역의 숨결이자 세상을 바라보는 창(窓)입니다. 제주어의 억양과 단어 속에는 제주의 거친 바람과 푸른 바다, 그리고 척박한 자연을 일궈낸 사람들의 지혜가 고스란히 퇴적되어 있습니다.

공공의 자산인 이 언어 데이터가 닫힌 창고에 머물지 않고, 디지털 세상에서 누구나 자유롭게 꺼내 쓸 수 있는 '살아 숨 쉬는 유산'이 되기를 바랍니다.

바라는 미래

저는 표준어가 싫습니다.
개인적으로 하나의 잣대와 기준만을 강요하는 문화 사대주의를 매우 경계합니다. 획일화된 모습보다는, '한국'이라는 하나의 울타리 안에서도 각 지역의 고유한 언어와 정체성이 온전히 존중받기를 바랍니다.

국내를 여행하더라도 마치 새로운 세계를 만나는 것처럼 다채롭고 풍요로운 문화적 경험이 굳건히 존재하길 원합니다. 이 작고 열린 사전이 제주어의 부활과 고유문화 보존에 조금이나마 기여하기를 진심으로 희망합니다.
